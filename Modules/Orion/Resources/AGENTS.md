# Orion resource guidance

Store Orion-only prototypes, locale, textures, RSI files, audio, maps, and UI resources here.

Keep paths consistent with the module resource root. Update every prototype, locale, sprite-state, and audio reference when moving a resource. Validate YAML and RSI metadata. Preserve source attribution and asset-specific licensing.

Do not create root `_Orion` resource paths.
