# Lavaland module guidance

This directory belongs to the Lavaland module. Keep Lavaland-specific code and resources within its declared projects and resource root.

Do not introduce Orion-only dependencies. When Orion needs to interact with Lavaland, prefer public contracts or a narrowly defined integration owned by the correct module.
