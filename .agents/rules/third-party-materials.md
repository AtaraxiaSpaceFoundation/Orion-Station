# Third-party materials

Verify source, revision, license, and attribution before importing external code, text, sprites, audio, maps, or prompts.

Repository membership and the Contributor License Agreement do not grant rights to third-party material. Do not silently relicense imported content. Preserve required notices, file-level licenses, `.license` files, and asset metadata.

For ports, record:

- source repository and exact revision;
- source PR family;
- files and assets used;
- original licenses;
- modifications made for this repository;
- target module and dependencies.

If licensing is ambiguous, stop the import and describe the missing permission. Do not guess that an apparently open repository permits the intended reuse.
