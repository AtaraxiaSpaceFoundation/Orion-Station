# Numeric and text input

For numeric input define:

- accepted syntax;
- decimal separator policy;
- integer versus decimal semantics;
- minimum and maximum;
- overflow behavior;
- rounding rule;
- normalized stored representation.

For text input define:

- trimming;
- empty value policy;
- maximum length;
- allowed characters or markup;
- case sensitivity;
- localization or display escaping;
- persistence and logging policy.
