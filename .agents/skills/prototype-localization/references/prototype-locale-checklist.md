# Prototype locale checklist

Check:

- entity name and description;
- suffix or category labels;
- marking names;
- construction and machine text;
- dataset entries;
- examine text;
- UI labels generated from prototype data;
- selector variables and entity grammar;
- fallback locale behavior;
- inherited text from abstract parents.

Search both prototype ID and locale key before concluding a string is unused.
