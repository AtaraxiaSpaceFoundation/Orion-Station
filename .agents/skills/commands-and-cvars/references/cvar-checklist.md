# CVar checklist

- stable namespaced key;
- typed default;
- documented unit;
- valid bounds;
- correct server/client/archive/replication flags;
- runtime update behavior defined;
- change subscription cleaned up;
- config compatibility considered;
- dangerous values rejected before use.
